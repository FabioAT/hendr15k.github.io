Anleitung
  • Downloaden sie sich das Passende Image fuir ihr Raspberry von der Projektseite https://retropie.org.uk/download/. 
  • Brennen sie dieses Image auf eine SD-Karte empfohlen fuir diesen Schritt wird das Programm Etcher zufinden aufhttps://www.balena.io/etcher/. 
  • Installieren sie sich notepad ++: https://ninite.com/notepadplusplus/ninite.exe
  • Nach den Brennen gehen sie in die Partition boot die auf den Datentraeger erstellt wurde und erstellen eine leere Datei mit den nahmen ssh. 
  • Oeffnen sie in der Bootpartition die datei config.txt mit notepad ++.
  • Entfernen sie das # vor sdtv_mode=2.
  • Loeten sie den cinch Display an pp6 - (ground, Ausere draht des Cinch Kabel) und pp24 +(Innere Draht des Cinch Kabel).
  • Stecken sie den Raspberry ans Lan und an den Strom. 
  • Installieren sie sich auf ihren Pc PuTTY hier zu downloaden https://ninite.com/putty/. 
  • Verbinden sie den pi mit einem Netzwerk
  • Gehen sie in den Windows Explorer und gib in die Datenpfardleiste "\\retropie" ein nun koennen sie die roms in den Ordner roms reinschieben.
  • Verbinden sie sich per Putty oder anderen ssh Programmen auf den SSH Port 22 Hostname: retropie.
  • SSH Benutzername: pi Passwort: raspberry
  • Geben sie die folgenden Befehle ein: 
    cd ~
    git clone https://github.com/mholgatem/GPIOnext.git
    bash GPIOnext/install.sh
  • gpionext config
  • Richten sie nun die Tasten als Tastertur ein.
  • Geben sie nun ins Terminal: "gpionext start" ein
  • Die Tasten koennen nun in retropie eingerichtet werden
