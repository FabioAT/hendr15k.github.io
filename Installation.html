Installation

 
Übersicht
  • Downloaden sie sich das Passende Image für ihr Raspberry von der Projektseite https://retropie.org.uk/download/. 
  • Brennen sie dieses Image auf eine SD-Karte empfohlen für diesen Schritt wird das Programm Etcher zufinden auf https://www.balena.io/etcher/. 
  • Nach den Brennen gehen sie in die Partition boot die auf den Datenträger erstellt wurde und erstellen eine leere Datei mit den nahmen ssh.
  • Stecken sie den Raspberry ans Lan und an den Strom.
  • Installieren sie sich auf ihren Pc PuTTY hier zu downloaden https://ninite.com/putty/.
  • Gehen sie in den Windows Explorer und gib in die Datenpfardleiste "\\retropie" ein nun können sie die roms in den Ordner roms reinschieben.
